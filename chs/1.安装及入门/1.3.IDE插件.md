## 1.3. IDE插件

现代化的编程语言一般都会为各种IDE和编辑器提供扩展，以提高编程的体验。凹语言为 VS Code、Fleet 和 Vim 提供了基本插件支持。

### 1.3.1 VS Code 插件

在 VS Code 扩展商店检索 “wa” 即可以查到凹语言插件。安装之后会有基本的语法高亮等功能。
